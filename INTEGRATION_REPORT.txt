🚀 كيفية البدء
═══════════════════════════════════════════════════════════════════════════

1. الانتقال إلى مجلد المشروع:
   cd /home/ubuntu/asserag-clone

2. تثبيت التبعيات (إذا لم تكن مثبتة):
   pnpm install

3. بدء خادم التطوير:
   pnpm run dev

4. بناء المشروع للإنتاج:
   pnpm run build

5. معاينة البناء:
   pnpm run preview
